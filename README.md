## @initx-plugin/cp

Copy plugin for `initx`

## Usage

```bash
npm install @initx-plugin/cp -g
```

Copy some text to clipboard

### Current Working Directory

```bash
npx initx cp cwd
```

### SSH Public Key

```bash
npx initx cp ssh
```

### GPG Public Key

```bash
npx initx cp gpg
```
